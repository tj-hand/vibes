# Vibes

CSS/Styles library module for Spark stack (frontend only).

## Overview

Vibes provides shared CSS styles, design tokens, and UI components for the Infinity frontend.

## Structure

```
vibes/
└── vibes-fe/           # Frontend components
    ├── styles/         # CSS/SCSS files
    └── components/     # Shared UI components
```

## Installation

This module is deployed via Spark:

```bash
./spark deploy vibes
```

## Dependencies

- No dependencies (frontend only)
